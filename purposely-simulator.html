Charity Board Self-Assessment

This RUT helps directors assess their knowledge and compliance with the Code of Governance for Charities and IPCs

  1. The RUT will generate a scenario, using publicly available resources in the Singapore charity sector.
  2. Based on the generated scenario, the relevant paragraphs from the Governance Evaluation Checklist (GEC) will be shown.
  3. The director will be prompted if the charity has put this principle into action: Yes or No or Partial to deal with the scenario.
  4. The director will be asked if another scenario is needed and if yes generate again.
  5. If no, the RUT will end the session.
  6. The session details can be shared via WhatsApp.
